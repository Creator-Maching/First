Heimer Decks

This page demonstrates my front-end skills. It features decks of my favorite champion from Legends of Runeterra, Heimerdinger. By clicking the links above, you’ll be redirected to a famous deck-building site (Mastering Runeterra) where you can view these decks. All lists include Heimerdinger and an alternating allied champion.

Madness Minds

This list features Viktor as an ally. The deck focuses on maximizing the effects of keywords. Heimerdinger provides robots with unique abilities, each with a different keyword, while Viktor gains random ones. To finish the game, use high-impact spells like "Glorious Evolution," "Give It All," or "Purrsuit of Perfection."

Master Minds

This deck pairs Heimerdinger with Jayce. The main strategy is to build a strong board using robots and casting 6+ cost spells to level up Jayce. At level 2, Jayce grants "Quick Attack" to all allies. Heimerdinger’s 6-cost robot has "Elusive," making it harder for opponents to counter.

Celestial Tech

In this deck, Aphelios provides lunar weapons for additional benefits, while Heimerdinger generates 2-cost tech robots. The key card here is "The Veiled Temple," which recharges mana and turns Aphelios into a late-game finisher.

Yordles

This deck features Norra, an exclusive champion from Legends of Runeterra. The strategy revolves around portals that summon random units. Unlike other Heimer decks focused on keywords, this one sacrifices quality for quantity, creating a swarm of units to overwhelm opponents. Key finisher: "Sneezy Biggledust!"
